Project 1

Image recognition app using a Machine Learning API. It has a frontend using React. It has its own server using Node.js and Express.js. It has PostgreSQL database where all registered users are stored securely with passwords in the form of hashes. When you enter an image and press detect, it will actually detect faces in the images and put a box around it.
